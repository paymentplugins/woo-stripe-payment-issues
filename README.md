# Payment Plugins for Stripe WooCommerce Issues
  This repository is used to track bug reports, feedback, feature requests, and compatibility issues
  for the [Payment Plugins for Stripe WooCommerce](https://wordpress.org/plugins/woo-stripe-payment/)
  plugin by Payment Plugins.

  ## Before Submitting an Issue

  - Ensure you are running the latest version of the plugin
  - Check if the issue has already been reported by searching existing issues
  - Deactivate all other plugins to rule out a conflict, then re-test

  ## Submitting an Issue

  Click **New Issue** and select the appropriate template.

  ## Other Support Channels

  - [Documentation](https://paymentplugins.com/documentation/stripe)
  - [WordPress.org Support Forum](https://wordpress.org/support/plugin/woo-stripe-payment/)

  ## Note

  This repository contains no plugin code. The plugin is developed and maintained
  in a private repository.
